# Standard Project Phases

Every civil engineering land development proposal uses these five phases, in this order:

1. **Prepare** — Project initiation, site visits, data collection, surveying, existing conditions review
2. **Schematic Design** — Early concept sketches, feasibility studies, preliminary layouts
3. **Concept Design** — Refined design options, client review, regulatory pre-consultation
4. **Detailed Design** — Full engineering drawings, specifications, permit submissions, final design packages
5. **Contract Administration** — Tendering support, construction oversight, site inspections, as-builts, project closeout

## Mapping Rules

- If a scope item involves data gathering, research, or setup work → **Prepare**
- If a scope item involves early layouts or feasibility → **Schematic Design**
- If a scope item involves design options or client review → **Concept Design**
- If a scope item involves final drawings, calculations, or permit applications → **Detailed Design**
- If a scope item involves construction support, inspections, or closeout → **Contract Administration**
- A single scope item can span multiple phases if it runs through the project lifecycle
